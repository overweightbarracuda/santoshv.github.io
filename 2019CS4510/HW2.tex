\problem{1. XOR}{2}
Construct a DFA that represents the XOR of two languages $L_1$ and $L_2$.

Construct a minimal DFA that represents the XOR of two languages $L_1$ (bitstrings of
even length) and $L_2$ (bitstrings with odd number of 1’s).

\problem{2. Integers and Floats}{2}
Construct a DFA that accepts integer/float numbers.

An integer number is defined as:
\begin{itemize} \itemsep -2pt
    \item A string leading with an optional + or -
    \item Followed by a single zero or non-empty sequence of digits that doesn’t start with a zero
\end{itemize}
A float number is defined as:
\begin{itemize} \itemsep -2pt
    \item A string leading with an optional + or -
    \item Followed by a single zero or non-empty sequence of digits that doesn’t start with a zero
    \item Followed by a decimal point
    \item Followed by a non-empty sequence of digits
\end{itemize}

\problem{3. Reverse Concatenation}{2}
Given a DFA that accepts some set $S$ of strings (possibly infinitely many) from a fixed
alphabet, construct a DFA that accepts all strings that are of the form $ab$ where $a$ is
a string in $S$ and $b$ is the reverse of some string in $S$.
